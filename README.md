## 추가 기능
1,2학년 복도 큰 모니터에 실시간으로 특별실 신청확인  
신청인 이름 입력하지 않고 버튼으로 선택기능  
선생님이 생성한 상점 항목 신청 기능   
학생은 자기 상점&벌점 조회만 가능(수정 불가)  
상점 순위전체 공개  
벌점에 따라 퇴사 여부 표시  
선생님이 해당 항목 클릭해서 벌점&상점 부여 (다중선택가능)  
엑셀파일로 상&벌점 출력가능
